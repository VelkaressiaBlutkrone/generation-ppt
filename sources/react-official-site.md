# 1
https://react.dev
React 공식 문서 사이트
UI 구축을 위한 JavaScript 라이브러리. Learn(학습), Reference(API 참조), Community(커뮤니티), Blog(공식 소식)로 구성. 컴포넌트 기반, 선언적 UI, Learn Once Write Anywhere 철학.

# 2
https://react.dev/learn
Learn React - Quick Start
일상적으로 사용하는 React 개념 80% 소개. 컴포넌트 생성/중첩, JSX 마크업, 스타일 추가, 데이터 표시, 조건부 렌더링, 리스트 렌더링, 이벤트 처리, useState로 화면 업데이트, Hooks 사용, 컴포넌트 간 데이터 공유(lifting state up). 튜토리얼(틱택토) 링크 포함.

# 3
https://react.dev/learn
https://react.dev/learn/installation
Learn + Installation
Quick Start와 설치 가이드. React는 점진적 도입 설계. Try React(샌드박스, CodeSandbox, StackBlitz), Creating a React App(프레임워크 권장), Build from Scratch, Add to existing project. Create React App은 deprecated.

# 4
https://react.dev/learn/creating-a-react-app
Creating a React App
새 React 앱 시작 시 프레임워크 권장. Full-stack: Next.js(App Router), React Router(v7)+Vite, Expo(네이티브). CSR, SPA, SSG 지원. React Server Components, Suspense 데이터 페칭. Vite, Parcel, RSbuild로 from scratch 빌드 가능.

# 5

https://react.dev/learn/creating-a-react-app
설치 및 앱 생성
설치 없이 샌드박스로 체험, HTML 페이지 다운로드로 로컬 시도. npx create-next-app@latest, npx create-react-router@latest, npx create-expo-app@latest. Create React App 대신 Vite/Parcel/RSbuild 권장.

# 6
https://react.dev/learn/tutorial-tic-tac-toe
Tutorial: Tic-Tac-Toe
첫 React 미니앱 구축 튜토리얼. 컴포넌트, props, state, 이벤트 핸들러, 조건부 렌더링, 리스트와 key, 불변성, 시간 여행(히스토리) 등 핵심 개념을 틱택토 게임으로 학습.

# 7

React Reference

# 8
https://react.dev/reference/react/hooks
React Hooks
컴포넌트에서 React 기능 사용. State(useState, useReducer), Context(useContext), Ref(useRef, useImperativeHandle), Effect(useEffect, useLayoutEffect, useInsertionEffect, useEffectEvent), Performance(useCallback, useMemo, useDeferredValue, useTransition), 기타(useSyncExternalStore, useId, useDebugValue, useActionState). 커스텀 Hook 정의 가능.
