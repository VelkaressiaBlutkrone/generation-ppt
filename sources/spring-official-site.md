# 1
https://spring.io
Spring 공식 메인 홈페이지
Java를 simple, modern, productive, reactive, cloud-ready하게 만드는 Spring 프레임워크 소개. Why Spring, Quickstart 링크 제공. Generative AI, Microservices, Reactive, Cloud, Web apps, Serverless, Event Driven, Batch 등 Spring이 할 수 있는 기능들 소개. Spring Boot 코드 예제와 Netflix 인용문 포함.

# 2
https://spring.io/why-spring
Why Spring 페이지
Spring이 Java 프로그래밍을 더 빠르고, 쉽고, 안전하게 만드는 이유 설명. Spring is everywhere(전 세계 개발자 신뢰), flexible(IoC/DI 기반), productive(Spring Boot, Spring Cloud), fast(빠른 시작/종료, LiveReload), secure(Spring Security), supportive(커뮤니티, 가이드, 이벤트) 섹션으로 구성.

# 3
https://spring.io/quickstart
Spring Quickstart 가이드
Hello World 엔드포인트를 만드는 단계별 튜토리얼. Step 1: start.spring.io로 웹 프로젝트 생성, Step 2: DemoApplication.java에 @RestController, @GetMapping 코드 추가, Step 3: gradlew bootRun으로 실행. IDE(IntelliJ, VS Code, Eclipse), JDK 17/21 필요. RESTful Web Service, Consuming REST, JPA 가이드 링크.

# 4
https://spring.io/projects/spring-boot
Spring Boot 프로젝트 페이지
stand-alone, production-grade Spring 애플리케이션을 "just run"할 수 있게 하는 프레임워크. 코드 생성 없음, XML 불필요, 메트릭/헬스체크, 자동 설정, starter 의존성, Tomcat/Jetty/Undertow 내장, fat jar 등 특징. Getting Started, Talks and videos, Spring Initializr 링크.

# 5
https://spring.io/ai
Spring AI (Generative AI) 페이지
Spring 애플리케이션에 Gen AI 기능을 통합. Generative AI 정의(텍스트/이미지/비디오 생성, RAG, 멀티모달). Spring AI는 ChatClient, Tool Calling, MCP(Model Context Protocol), RAG 지원. OpenAI, Microsoft, Amazon, Google, Hugging Face 등 모델 제공자 및 벡터 DB 지원.

# 6
https://spring.io/microservices
Microservices 페이지
마이크로서비스 아키텍처 소개. Spring Boot로 작게 시작하고 빠르게 반복. Spring Cloud로 서비스 디스커버리, 로드밸런싱, 서킷브레이커, 분산 추적. Spring Cloud Stream으로 이벤트 기반 시스템. Micrometer로 메트릭, Prometheus 연동. Cloud Foundry 배포.

# 7
https://spring.io/reactive
Reactive 페이지
비동기, 논블로킹, 백프레셔 지원 리액티브 시스템. Project Reactor 기반. Spring WebFlux, Spring Data 리액티브 리포지토리. MongoDB, Redis, Cassandra, R2DBC(Postgres, SQL Server 등) 리액티브 지원. 적은 리소스로 더 많은 동시 연결 처리.

# 8
https://spring.io/cloud
Spring Cloud 페이지
분산 시스템 개발 지원. 12-factor 앱 이슈(외부 설정, 무상태, 로깅). Service discovery(Eureka, Consul, Zookeeper, Kubernetes), API Gateway, Cloud Config(Git 연동), Circuit Breaker(Resilience4J, Sentinel, Hystrix), Micrometer Tracing(Zipkin, OpenTelemetry), Spring Cloud Contract(계약 기반 테스트).

# 9
https://spring.io/web-applications
Web Applications 페이지
웹 애플리케이션 빠른 구축. Spring Boot로 최소 설정, 내장 서버, 메트릭/헬스체크. Spring Security로 SAML, OAuth, LDAP, OWASP Top 10 방어. Spring Data로 JPA, MongoDB 등 데이터 스토어 연결. Java, Kotlin, Groovy 지원.

# 10
https://spring.io/guides
Spring Guides 페이지
Spring 팀이 권장하는 최신 프로젝트와 기법으로 빠르게 생산성을 높이는 가이드 모음. REST APIs, Data Access, Security, Messaging, Microservices, Testing, Cloud Platform 등 카테고리별 가이드. 예상 소요 시간(30m, 1h, 3h) 필터 제공.
