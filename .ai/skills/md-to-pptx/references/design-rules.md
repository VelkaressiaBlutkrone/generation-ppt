# PPT 디자인 규칙 (PPT_Design_Guide_2026 기반)

PPTX 생성 시 자동 적용해야 할 핵심 규칙만 추출한 레퍼런스.

## 슬라이드 기본

- **크기**: 16:9 와이드스크린 (13.333" × 7.5")
- **여백**: 상하좌우 최소 0.5~0.8인치, 면적의 30%+ 여백 확보
- **요소 간 간격**: 최소 16px (8px 그리드 시스템)
- **핵심 메시지**: 슬라이드 중앙부 70% 영역 안에 배치

## 타이포그래피

| 요소 | 크기 | 굵기 |
|---|---|---|
| 슬라이드 제목 | 48~120pt | Bold/ExtraBold |
| 섹션 제목 | 32~48pt | Bold |
| 핵심 메시지 | 24~32pt | Bold |
| 본문 | 20~28pt (최소 16pt) | Regular/Medium |
| 캡션/출처 | 12~14pt | Regular |

- **폰트**: 최대 2종 (Pretendard 권장 — 한영 모두 커버)
- **행간**: 1.4~1.6배
- **자간**: -5~0%
- **Thin/Light 계열 본문 금지**

## 색상 (60-30-10 법칙)

```
60% → 주색 (배경, 여백)
30% → 보조색 (섹션 구분, 카드 배경)
10% → 강조색 (CTA, 핵심 수치)
```

- 텍스트-배경 대비: WCAG AA 4.5:1 이상
- 색맹 지원: Red-Green 대비 대신 Blue-Orange 계열
- 색상만으로 정보 전달 금지 (형태·텍스트 병행)

## 표 디자인

- 세로선 완전 제거
- 가로선: 헤더 아래 1개만, 나머지 아주 연하게
- 헤더: 배경색 또는 볼드 처리
- 숫자: 우측 정렬 + 소수점 자리 맞춤
- 텍스트: 좌측 정렬
- 셀 패딩: 최소 12px
- 최대 6열 권장

## 차트

- 3D 효과/그림자 절대 금지
- 파이 차트 사용 금지 (도넛으로 대체)
- 범례 대신 Direct Labeling
- 기본 데이터: 회색 계열
- 강조 포인트: 브랜드 컬러 1개
- Y축 기준선 반드시 0 (막대 차트)
- 최대 선 3개 (꺾은선 차트)

## 이미지

- 품질: 최소 1920×1080px
- 톤 통일: 동일 필터·채도
- Full Bleed 시 텍스트 가독성 확인
- Alt 텍스트 필수

## 아이콘

- 라인 두께 통일 (1.5px 또는 2px)
- 스타일 통일 (Filled/Outlined/Duotone 중 하나)
- 크기: 24/32/48px 중 일관
- 컬러: 본문 색상 또는 강조색 1가지
